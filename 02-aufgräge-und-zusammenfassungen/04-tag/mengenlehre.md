# Mengenlehre – Grundlagen

| Symbol | Bedeutung | Beispiel |
|--------|-----------|----------|
| ∈ | Element von | a ∈ A |
| ∉ | Nicht in Menge | b ∉ A |
| {} / Ø | Leere Menge | A = {} |
| ⊂ / ⊆ | Teilmenge | B ⊂ A |
| ∩ | Schnittmenge | A ∩ B |
| ∪ | Vereinigungsmenge | A ∪ B |
| \ | Differenzmenge | B\A |
| Xc | Komplement | Ac |

Diese Logik ist auch auf SQL-JOINs übertragbar:
- `INNER JOIN` = Schnittmenge (∩)
- `LEFT JOIN` = A ∪ B mit NULL für fehlende B-Werte
